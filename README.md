# Black-box Testing Practice

Dự án thực hành kiểm thử hộp đen cho các bài toán lập trình cơ bản.

## Các bài toán đã thực hiện
1. Chu vi/Diện tích HCN
2. Giải phương trình bậc 2
3. Tính ngày trong tháng
4. Kiểm tra số nguyên tố
5. Tính tổng chuỗi S1, S2
6. Tìm UCLN

## Kỹ thuật kiểm thử áp dụng
* **Phân lớp tương đương:** Chia dữ liệu thành các nhóm hợp lệ và không hợp lệ.
* **Phân tích giá trị biên:** Kiểm tra tại các ngưỡng đặc biệt (ví dụ: tháng 2 năm nhuận, hệ số a=0).
